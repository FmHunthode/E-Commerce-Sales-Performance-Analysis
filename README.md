# E-Commerce-Sales-Performance-Analysis
## Project Overview
This project analyzes e-commerce sales transactions from 2022 to 2024 to evaluate business performance across products, categories, regions, and seasonal periods.
The objective was to identify revenue drivers, profit trends, regional performance patterns, and seasonal opportunities that can support data-driven business decisions.
The analysis was conducted entirely in Microsoft Excel using Pivot Tables, calculated fields, charts, and interactive dashboard components.
## Dataset Overview
The dataset contains transactional sales records from an e-commerce business.
### Dataset Summary
| Metric              | Value        |
| ------------------- | ------------ |
| Total Orders        | 3,500        |
| Total Revenue       | $10.67M      |
| Total Profit        | $1.84M       |
| Total Quantity Sold | 17,261 Units |
| Period Covered      | 2022 – 2024  |

## Data Cleaning and Preparation

The following preparation steps were carried out before analysis:

● Verified data consistency.

● Converted date fields into Year and Month dimensions.

● Created Profit Margin calculations.

● Checked for missing values.

● Validated numerical fields.

● Structured the dataset for Pivot Table analysis.
## Analytical Approach

The analysis was divided into three major business perspectives:

### 1. Overall Performance Analysis
● Revenue

● Profit

● Quantity Sold

● Orders received

● Average Order Value

### 2. Seasonal Trend Analysis
● Monthly revenue and profit patterns

● Year-over-year performance

● Seasonal demand fluctuations

### 3. Regional Performance Analysis
● Revenue by region

● Profit contribution by region

● Product performance across regions

## Key Questions

 The project aimed to answer the following business questions:

● What is the overall revenue performance of the business?

● Which product categories generate the most revenue?

● Which regions perform best?

● How has performance changed across years?

● What opportunities exist for revenue growth?
